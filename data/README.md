# Data

## Dataset

This project uses NFL tracking data from the [**NFL Big Data Bowl 2025**](https://www.kaggle.com/competitions/nfl-big-data-bowl-2025/data). The dataset is unfortunately no longer publicly available on Kaggle.

## Data Description

- **games.csv** - Game-level information such as score and the teams playing
- **plays.csv** - Play-by-play data from each game 
- **players.csv** - Player information from players listed in any of the tracking files
- **player_play.csv** - Player-level stats for each play of each game
- **tracking_week_*.csv** - Player tracking data (coordinates, speed, etc.) of each player for each play at each point in time (measured in frames)

Tracking week files were not used in this analysis. For a more detailed description of the files and their variables, please visit the above link to the competition.

## Expected Data Structure

Once you have the data, organize files as follows:
```
data/
├── raw/
│   ├── games.csv
│   ├── plays.csv
│   ├── players.csv
│   └── player_play.csv
└── processed/
    └── (generated by preprocessing)
```

## Preprocessing

If you have the raw data, preprocess it by running:
```bash
python src/test.py
```

This will load the data, generate the engineered features used for model training, train the model and evaluate it.
